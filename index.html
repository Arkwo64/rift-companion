Rift Companion

A personal League of Legends analytics dashboard.
Personal project · runs locally · single account

Rift Companion aggregates my own Riot API match history and turns it into actionable insights to help me improve and make better pre-game decisions. It is a local web app (Python, standard library only) that I run on my own machine.

What it does

🏆 Champion performance

Best/worst champions using sample-size-adjusted win rates (Wilson score) cross-referenced with meta win rates for my elo.

⚔️ Lane matchups

My win rate against each enemy laner, split by enemy role (ADC, support, etc.).

🛠️ Build vs meta

The items and runes I actually build compared to the recommended meta builds.

👥 Premades & synergies

The duos I queue with and which champion pairings win most together.

⏰ Habits & tilt

Form by time of day and after losses, plus early-game leads and snowball/comeback tendencies from match timelines.

🔴 Live game

When I load into a match: both teams, each player's rank, and detected premades (uses the Spectator API).

Riot APIs used

Account-V1, Summoner-V4, League-V4, Match-V5, Spectator-V5, Champion-Mastery-V4, and Data Dragon. Low-traffic, personal use, a single account.
